## 🧠 About Me

I’m a Software Engineering student at ASTU who builds things that go beyond tutorials — real apps, real systems, real use cases.

I work across **frontend, full-stack, and mobile development**, and I enjoy connecting everything into one working system.

Most of what I build includes:
- 📱 Flutter mobile apps (offline-first with Hive, real-time features)
- 🌐 Full-stack systems (APIs + database + frontend)
- ⚡ FastAPI backends (clean, fast, and scalable)
- 🔥 Firebase (Firestore, Realtime DB, integrations)
- 🤖 IoT systems (ESP32 + mobile control)

Some projects start small… then suddenly I’m adding APIs, background services, and automation 😅
